# Sistema Contábil Integrado - Grupo 2

## Sobre o Projeto

Este projeto consiste em um sistema contábil com viés financeiro, desenvolvido para integrar de forma automática e transparente as movimentações financeiras com a contabilidade empresarial.

## Conceito

A principal característica do sistema é garantir que **todo lançamento financeiro reflita automaticamente na movimentação das contas contábeis** do plano de contas. Isso significa que cada transação financeira registrada no sistema gera, simultaneamente, os respectivos lançamentos contábeis nos débitos e créditos das contas apropriadas.

Esta abordagem elimina a necessidade de dupla digitação e reduz significativamente os riscos de inconsistências entre os registros financeiros e contábeis, proporcionando uma visão unificada e sempre atualizada da situação econômico-financeira da empresa.

## Arquitetura

O sistema é estruturado como um mono-repositório contendo:

- **Frontend**: Aplicação React com Vite e TypeScript para interface do usuário
- **Backend**: API REST desenvolvida em Django para processamento e armazenamento dos dados

## Objetivo

Oferecer uma solução robusta e confiável para empresas que precisam manter rigoroso controle contábil-financeiro, automatizando processos e garantindo a integridade dos dados através da sincronização automática entre movimentações financeiras e registros contábeis.
